Detailed Description

Renders Pinterest Board widget with board's latest pins.

Requires Pinterest scripts bundle to be included.

See also
http://business.pinterest.com/widget-builder/::do_embed_board, IWidgetsScriptsRendererExtensions.Pinterest(IWidgetsScriptsRenderer)
