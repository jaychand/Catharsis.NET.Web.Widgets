Detailed Description

Renders Twitter "Follow" button.

Requires Twitter scripts bundle to be included.

See also
https://dev.twitter.com/docs/follow-button, IWidgetsScriptsRendererExtensions.Twitter(IWidgetsScriptsRenderer)
