Detailed Description

Renders Cackle latest comments widget for registered website.

Requires Cackle scripts bundle to be included.

See also
http://ru.cackle.me/help/widget-api, IWidgetsScriptsRendererExtensions.Cackle(IWidgetsScriptsRenderer)
