Detailed Description

Renders Pinterest Profile widget with user's latest pins.

Requires Pinterest scripts bundle to be included.

See also
http://business.pinterest.com/widget-builder/::do_embed_user, IWidgetsScriptsRendererExtensions.Pinterest(IWidgetsScriptsRenderer)
