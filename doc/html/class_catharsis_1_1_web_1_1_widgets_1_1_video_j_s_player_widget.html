Detailed Description

Renders Video.JS web player widget.

Requires VideoJS scripts and CSS bundles to be included.

See also
http://www.videojs.com, IWidgetsScriptsRendererExtensions.VideoJS(IWidgetsScriptsRenderer), IWidgetsStylesRendererExtensions.VideoJS(IWidgetsStylesRenderer)
