Detailed Description

Renders Twitter "Tweet" button.

Requires Twitter scripts bundle to be included.

See also
https://dev.twitter.com/docs/tweet-button, IWidgetsScriptsRendererExtensions.Twitter(IWidgetsScriptsRenderer)
