Detailed Description

Renders Mail.ru "Like" button on web page.

Requires MailRu scripts bundle to be included.

See also
http://api.mail.ru/sites/plugins/share, IWidgetsScriptsRendererExtensions.MailRu(IWidgetsScriptsRenderer)
