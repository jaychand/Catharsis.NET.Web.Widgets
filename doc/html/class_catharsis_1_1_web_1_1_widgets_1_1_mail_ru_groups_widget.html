Detailed Description

Renders Mail.ru Group (People In Group) widget.

Requires MailRu scripts bundle to be included.

See also
http://api.mail.ru/sites/plugins/groups, IWidgetsScriptsRendererExtensions.MailRu(IWidgetsScriptsRenderer)
