Detailed Description

Renders Pinterest "Pin It" button widget.

Requires Pinterest scripts bundle to be included.

See also
http://business.pinterest.com/widget-builder/::do_pin_it_button, IWidgetsScriptsRendererExtensions.Pinterest(IWidgetsScriptsRenderer)
