Detailed Description

Renders Surfingbird "Surf" button.

Requires Surfingbird scripts bundle to be included.

See also
http://surfingbird.ru/publishers/surfbutton, IWidgetsScriptsRendererExtensions.Surfingbird(IWidgetsScriptsRenderer)
