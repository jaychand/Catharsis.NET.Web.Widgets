Detailed Description

Renders Mail.ru Faces (People On Site) widget.

Requires MailRu scripts bundle to be included.

See also
http://api.mail.ru/sites/plugins/faces, IWidgetsScriptsRendererExtensions.MailRu(IWidgetsScriptsRenderer)
